Percent Pipe

El uso de la coma o punto viene dado por el LOCALE_ID al igual que con los números y moneda local, es decir, si configuraste tu local en es-AR se utiliza coma en vez de punto para definir los decimales, sin embargo puedes alterarlo pasanadole el locale que quieres mostrar como parámetro

Más info
{{this.porcentaje | percent }}
{{this.porcentaje | percent:'2.2-2' }}
{{this.porcentaje | percent:'2.2-2':'en' }}

Importante:

para agregar traducción local a nuestros pipes, debemos agregar lo siguiente (omitiendo las comillas dobles) a nuestro módulo: 

    {{ "import { LOCALE_ID, NgModule } from '@angular/core';" }}
    {{ "import { registerLocaleData } from '@angular/common';" }}

    // importar locales
    import localeEsAr from '@angular/common/locales/es-AR';

    providers: {{"[{ provide: LOCALE_ID, useValue: 'es-Ar' }]" }},

para definir por defecto el pipe currency a nuestra moneda local agregamos lo siguiente (omitiendo las comillas dobles) a nuestro módulo: 

    {{ "import { DEFAULT_CURRENCY_CODE, LOCALE_ID, NgModule } from '@angular/core';" | json }}

    providers: {{"[{ provide: DEFAULT_CURRENCY_CODE, useValue: 'ARS' }]" | json }},

Recuerda que providers es un array y si ya tienes devinido el LOCALE_ID, agrega el nuevo provide como otro item
